# Source Code

This directory contains the source code for the Isaac robot system.

## Structure

- `vla_controller/` - Vision-Language-Action model implementation
- `hardware_drivers/` - Hardware driver wrappers and ROS 2 interfaces
- `control_modes/` - Control mode implementations
- `utils/` - Shared utilities and helper functions

## ROS 2 Workspace

This code is intended to be integrated into the ROS 2 workspace at `~/ros2_ws/src/`.

## Development

1. Create ROS 2 packages in appropriate directories
2. Follow ROS 2 package structure conventions
3. Document code with docstrings and comments
4. Add tests for critical functionality
