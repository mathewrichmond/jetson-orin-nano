# Realsense Camera Hardware Setup

This directory contains setup scripts and configuration files for Intel Realsense cameras.

## Files

- `install_realsense.sh` - Installation script for Realsense SDK and ROS 2 wrapper
- `config/` - Camera configuration files
- `launch/` - ROS 2 launch files
