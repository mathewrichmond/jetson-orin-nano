# Monitoring Infrastructure

This directory contains monitoring infrastructure components for system, hardware, and performance monitoring.

## Structure

- `system/` - System resource monitoring (CPU, memory, disk, network)
- `hardware/` - Hardware health monitoring (temperatures, power, devices)
- `performance/` - Performance metrics and analysis

## Components

- ROS 2 monitoring nodes
- Health check scripts
- Performance data collection
- Alert and notification system
