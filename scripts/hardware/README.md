# Hardware Installation Scripts

This directory contains scripts for installing and configuring hardware components.

## Scripts

- `install_realsense.sh` - Install Intel Realsense SDK and ROS 2 wrapper
- `install_motor_controller.sh` - Install motor controller drivers (TBD)
- `setup_raspberry_pi.sh` - Setup Raspberry Pi sub-module (TBD)

## Usage

Run scripts with sudo:
```bash
sudo ./scripts/hardware/install_realsense.sh
```
