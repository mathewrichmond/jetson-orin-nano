# Configuration Files

This directory contains configuration files for the Isaac robot system.

## Structure

- `system/` - System-wide configurations
- `hardware/` - Hardware-specific configurations
- `control/` - Control mode configurations

## File Formats

- YAML: Preferred for most configurations
- JSON: For structured data
- INI: For simple key-value configs

## Usage

Configuration files are loaded by:
- ROS 2 launch files
- Python/C++ applications
- System scripts
